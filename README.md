# WebGL Fluid Simulation

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

A WebGL fluid simulation that works in mobile browsers.

## Demo
[Play here](https://code4fukui.github.io/WebGL-Fluid-Simulation/) forked from [PavelDoGreat/WebGL-Fluid-Simulation](https://github.com/PavelDoGreat/WebGL-Fluid-Simulation)

## Features
- Fluid simulation using WebGL
- Supports mobile devices
- Customizable parameters like density/velocity dissipation, pressure, vorticity, splat radius, etc.
- Color customization
- Random splat generation

## Requirements
None. This is a client-side WebGL application.

## Usage
Open the HTML file in a web browser. The application will start running.

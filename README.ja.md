# WebGL Fluid Simulation

WebGLを使った流体シミュレーションのデモです。ブラウザ上で動作し、モバイルブラウザでも動作します。

## デモ
[こちら](https://code4fukui.github.io/WebGL-Fluid-Simulation/)でデモを試すことができます。

## 機能
- 流体シミュレーション
- カラー表示
- パフォーマンス調整
- GUI操作

## 使い方
ブラウザで開くと、自動的にシミュレーションが開始されます。GUIから各種設定を変更できます。

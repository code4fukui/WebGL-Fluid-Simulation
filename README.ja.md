# WebGL Fluid Simulation

モバイルブラウザで動作するWebGL流体シミュレーションです。

## デモ
[こちらで遊べます](https://code4fukui.github.io/WebGL-Fluid-Simulation/) （[PavelDoGreat/WebGL-Fluid-Simulation](https://github.com/PavelDoGreat/WebGL-Fluid-Simulation) からフォーク）

## 特徴
- WebGLを使用した流体シミュレーション
- モバイルデバイス対応
- 密度/速度の減衰、圧力、渦度、スプラット半径などのカスタマイズ可能なパラメータ
- カラーカスタマイズ
- ランダムなスプラット生成

## 必要条件
ありません。これはクライアントサイドのWebGLアプリケーションです。

## 使い方
HTMLファイルをウェブブラウザで開いてください。アプリケーションが起動します。
